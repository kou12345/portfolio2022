IDE

GitHub Codespaces

コードリポジトリから呼び出せるクラウド上のIDEおよび開発環境「GitHub Codespaces」において、個人ユーザーが1カ月に最大60時間まで無料で利用できるようになったことが発表された。
制限時間は、2コア環境の場合が60時間で、4コアで30時間、8コアで15時間。16コア以上は対象とはならない。
